Open Source Tools

Software Bill of Behavior

SBoB is the runtime addition to the Software Bill of Materials (SBoM). It allows the intended behavior of software to be transferred from the vendor to the end user — enabling defenders to maintain up-to-date detection rules and achieve genuine anomaly detection.

We are creating this standard especially for the EU Cyber Resilience Act and NIS 2 regulation, to help defenders get ahead of breaches.
